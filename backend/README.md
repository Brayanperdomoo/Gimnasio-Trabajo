# Gimnasio Backend

API PHP con CRUD completo para 5 entidades:

1. `entrenadores`
2. `planes`
3. `miembros`
4. `clases`
5. `reservas`

Cada entidad tiene 6 operaciones:

- `GET /api/recurso`
- `GET /api/recurso/{id}`
- `GET /api/recurso/filter?search=texto`
- `POST /api/recurso`
- `PUT /api/recurso/{id}`
- `DELETE /api/recurso/{id}`

Swagger: http://localhost:8080/docs
