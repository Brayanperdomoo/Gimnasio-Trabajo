# Repo Frontend - FitAdmin Gimnasio

Frontend en HTML, CSS, JavaScript y Axios para el proyecto de gimnasio.

## Páginas

- `index.html`: landing page con documentación visual del proyecto.
- `crud.html`: CRUD visual conectado al backend.
- `assets/js/app.js`: conexión Axios y lógica de los 30 servicios CRUD.
- `assets/css/styles.css`: diseño moderno inspirado en gimnasio/fitness.

## Ejecutar

```bash
docker compose up --build
```

El frontend queda en:

```text
http://localhost:3000
```

El backend esperado está en:

```text
http://localhost:8080/api
```
